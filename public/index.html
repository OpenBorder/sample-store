    🔌 How this checkout works (Open Border)

    This store never sees a card or a payment-processor key — it only uses two Open Border packages:
    1. @open-border/js — the browser embed. Loaded from the CDN, authenticated with the publishable key (pk_). It collects the card and returns a token (pm_). The product currency you pick above tells it which Open Border entity to route to.
    2. @open-border/node — the backend SDK. Our server takes that pm_, quotes duties & taxes for the ship-to address (createTaxQuote), then calls createPaymentIntent with the secret key (sk_) to make the charge.
